# Pet Adoption System – Future Improvements

1. **GUI Integration** – Use JavaFX to make a graphical interface.  
2. **Database Storage** – Replace CSV files with MySQL, SQLite, or MongoDB.  
3. **User Authentication & Roles** – Add Admin, Adopter, and Foster Home roles.  
4. **Advanced Compatibility** – Allow custom matching weights and preferences.  
5. **Reporting & Analytics** – Adoption statistics, popular breeds, time-to-adopt metrics.  
6. **Notification System** – Email alerts for adoption updates.  
7. **Mobile/Web Version** – Extend the system to web or mobile platforms.  
8. **Unit Testing** – Add JUnit tests for all modules.
